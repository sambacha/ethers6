Wordlists
=========

*This is a beta version of the next release of ethers. Please do not use it in production yet.*

This package contains well-established BIP-39 wordlists for various
languages.

Available languages:

- `cz` - Czech
- `en` - English
- `es` - Spanish
- `fr` - French
- `ja` - Japanese
- `ko` - Korean
- `pt` - Portuguese
- `zh_cn` - Simplified Chinese
- `zh_tw` - Traditional Chinese
